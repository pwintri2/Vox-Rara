# Wintrip Ebook Promo Co-Pilot

Lokale promotie-assistent voor het Engelstalige e-boek `The Soul Journey of John May`, te koop via `https://wintrip-projects.nl`.

Open `index.html` in je browser. De app:

- toont de meegeleverde video als upload-asset;
- maakt per kanaal een tekstvariant en UTM-link;
- opent veilige deelvensters of uploadpagina's;
- houdt bij welke posts klaar zijn;
- maakt een mini-poster met QR-code;
- exporteert een compact promotieplan.

De tool publiceert niet automatisch namens jou. Jij logt zelf in, kiest de juiste context en plaatst handmatig.
